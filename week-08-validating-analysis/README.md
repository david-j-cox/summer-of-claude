# Week 8 - Validating Analysis

Status: outline only - to be built out. This is the lab's core week.

## Objectives
The questions students must make their agent ask:
- What are this test's assumptions, and does the data meet each one?
  (normality, homogeneity of variance, independence, sphericity, ...)
- Is there a more appropriate or more modern method? Why this one over it?
- Right multiple-comparison correction? Right effect size? Right model for the design?
- Where could data leakage, silent type coercion, or a wrong default change the result?
- Reproducibility: seeds, environments, deterministic pipelines.
- Watch for hallucinated methods/citations and p-hacking-by-autocomplete.
