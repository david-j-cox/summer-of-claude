# Summer of Claude

A 10-week summer series for the lab on building with AI coding tools
(Claude Code and Codex), and just as important, learning to question, test, and
validate what they produce so the outputs can be trusted.

New here? Start with **[Week 1](week-01-setup-and-first-build/)** to get set up.

## Getting the files

```
git clone https://github.com/david-j-cox/summer-of-claude.git
cd summer-of-claude
```

To grab updates later: `git pull`.

## What's here

- `Summer of Claude.docx`: the full syllabus (one-page overview of all 10 weeks).
- `week-01` ... `week-09`: one folder per week with materials and exercises.
- `skills/`: take-home skills you can drop into your own projects.
- `resources/`: shared datasets, references, and reading.

## The two throughlines

- **Vocabulary:** demystify the jargon (agent, harness, tools, context, MCP,
  skills, slash commands, output styles, subagents, hooks) as it becomes useful.
- **Trust:** never accept generated code or analysis at face value. Learn the
  questions to ask, and encode them into skills so the agent asks them every time.
