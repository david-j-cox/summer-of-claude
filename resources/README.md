# Shared Resources

Cross-week references, datasets, and supplemental reading.

## Available now
- `glossary.md`: running glossary of all the jargon, grouped by the week each
  term first appears. Grows every week; Week 3 is a full pass back over it.
- `datasets/behavior_sessions.csv`: synthetic multilevel behavioral dataset used in
  Weeks 5, 8, and 10. See `datasets/DATA_DICTIONARY.md` for columns and structure,
  and `datasets/generate_behavior_sessions.py` to regenerate it (standard library
  only, fixed seed).
