# Take-Home Skills

Downloadable skills students collect across the series. Each is "a question (or a
job) encoded as a skill" so the agent does it every time without the student
having to remember.

## Planned
- `/explain` - repo tutor: explain any file/function/architecture at a chosen
  depth, trace data flow, quiz the student. (Week 6)
- `/check-assumptions` - before reporting any statistical result, state the test's
  assumptions, check each against the data, flag violations, suggest better
  methods. (Week 6)
- More added in Weeks 7-9.

## How to use
Each skill ships as a folder students can drop into their own project (Claude
Code) or adapt as a prompt/AGENTS.md snippet (Codex). Install instructions live
with each skill once built.
